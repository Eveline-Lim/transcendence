User Story: Core Gameplay Loop
As a competitive player,
I want to log in, play a match against IA

Flow of Events
1. Connection (Auth)
- The user arrives at the landing page.
- The user authenticates successfully (via standard login or OAuth).
- System Interaction: Auth Service issues a JWT.

2. Gameplay (Game)
- The user chooses the level of the IA
- The user enters directly in the game against AI
- The user plays the game until one score reaches the winning threshold.
- System Interaction: Game Service handles real-time positions, AI Opponent Service handles IA actions
